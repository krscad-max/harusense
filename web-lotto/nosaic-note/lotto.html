추천 번호 생성

규칙:
• 최근 60회에서 10회 이상 나온 번호 중 2개 랜덤
• 최근 60회에서 5회 이상 나온 번호 또는 최근 3회에 나온 번호들 중에서 2개 랜덤
• 나머지 2개는 전체(1~45)에서 중복 없이 채움. 결과는 오름차순 정렬.

데이터

최근 추첨 기록을 직접 편집할 수 있습니다. 편집 후 저장하면 로컬(브라우저 저장)으로 반영됩니다. draws.json 파일을 서버에 올릴 권한이 있으면 서버 업로드 후 사용 가능합니다.

저장 형식: JSON 배열. 각 항목은 {"round":,"nums":[n1,...,n6]} 형태 권장. (예: [{"round":1040,"nums":[1,2,3,4,5,6]}, ...])
