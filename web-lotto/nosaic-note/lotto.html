추천 번호 생성

규칙:
• 최근 60회에서 10회 이상 나온 번호 중 2개 랜덤
• 최근 60회에서 5회 이상 나온 번호 또는 최근 3회에 나온 번호들 중에서 2개 랜덤
• 나머지 2개는 전체(1~45)에서 중복 없이 채움. 결과는 오름차순 정렬.

데이터

최근 추첨 기록이 포함된 draws.json(최근 회차 배열)이 있으면 더 정확한 추천을 제공합니다. 기본적으로 샘플/시뮬레이션 데이터가 사용됩니다.

파일 형식: [[ [n1,n2,...,n6], ... ]] — 최신 회차가 배열의 마지막이어야 합니다.
