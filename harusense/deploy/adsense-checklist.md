AdSense 신청 체크리스트 — HaruSense

필수 준비물
- 사이트에 About / Contact / Privacy / Ads(광고 고지) 페이지가 존재할 것 (완료)
- 사이트에 최소 15개 이상의 고유 게시글(각 최소 600자 권장) — 현재 18개 확보
- 콘텐츠는 중복 없이 고품질이어야 함(오리지널 원고 권장)
- 사이트 디자인·네비게이션이 완전하고 광고 클릭 유도를 하지 않을 것
- 개인정보처리방침에 쿠키 및 광고 고지 포함(완료 초안 존재)

기술적 준비
- 모바일 친화적(Responsive) 디자인 적용(완료)
- robots.txt 및 sitemap.xml 준비(완료)
- SSL(HTTPS) 활성화 — Cloudflare가 자동 적용됨
- 중요한 페이지(About/Privacy/Contact) 접근 가능

신청 전 점검 항목
1. 각 글에 meta title, meta description 추가
2. 각 글에 내부 링크 1~2개 포함(관련글) — 체류 시간 향상
3. 페이지 로딩 속도 개선(이미지 WebP, lazy loading 적용) — 적용 완료
4. 광고 슬롯 레이아웃 미리 확보(상단, 본문, 사이드 중 최소 2곳 권장)
5. 애널리틱스(GA4 또는 Plausible) 연결 권장(심사 후 트래픽 모니터링)

애드센스 신청시 사용 가능한 문구(예시)
- 사이트 설명(간략): "HaruSense (하루센스)는 생활을 더 편하게 만드는 실용 팁과 절약 노하우를 제공하는 미디어입니다. 실용적인 아이디어와 검증된 노하우를 통해 일상 소비를 줄이고 삶의 효율을 높입니다."
- 대표 연락처: contact@harusense.example (실서비스 전 실제 이메일로 변경)

신청 절차(간단)
1. Google AdSense 계정 생성(또는 기존 계정 사용)
2. 사이트 정보 입력 및 사이트 URL 제출
3. AdSense에서 제공하는 코드(HTML 스니펫)를 승인 대기 중인 사이트의 <head>에 삽입(승인 전에는 광고가 노출되지 않음)
4. 승인이 완료되면 광고 코드를 원하는 슬롯에 삽입

주의사항
- 자동 클릭 유도, 클릭 유도 문구, 인센티브형 클릭 유도 금지
- 저품질 콘텐츠(짧은 복사본, 중복 콘텐츠)는 승인 거부 원인이 될 수 있음
